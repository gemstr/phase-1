Why use Version Control Anyway?

First of all forgive me for making this a very short blog. I am just trying to make this thing come to life. What are the benefits of version control? Version control is beneficial because it keeps track of all the changes we are going to be making to our codes. If we make a mistake, we can always go back to a version we started with and start over. How does git help you keep track of changes? Git saves the contents of the files in your project. It can reproduce the contents of the files exactly how you originally saved them. Any changes you or others make will be seen on Github. Why use Github to store your code? Why not? Honestly I don't know much about any other ways to store my code right now, I'd have to do some google investigating. As for now Github is all I know.
